Bootstrap Directory Structure

Understanding the purpose of CSS, JS, and Icons folders in Bootstrap

📁 CSS Folder

Contains files that define the grid system, layout utilities, and styles for all components.

  • bootstrap.css: Unminified file, good for development.
  • bootstrap.min.css: Minified file with spaces removed, optimized for speed.
📁 JS Folder

Contains scripts required for interactive elements like tooltips, modals, and accordions.

  • bootstrap.js: Core Bootstrap scripts.
  • bootstrap.bundle.js: Includes Popper.js inside for dropdowns and popups placement.
📁 Icons Folder

Bootstrap Icons are vector graphics (.svg or web fonts) used to add visual symbols to a web page.

  • Provides icon font files or individual SVG files.
  • Can be installed locally or loaded using a CDN link.
