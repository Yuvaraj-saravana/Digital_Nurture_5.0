Collapsible Accordion

A clean FAQ layout utilizing Bootstrap's accordion component.

Bootstrap 5 is the latest version of the world's most popular front-end open-source toolkit. It helps developers quickly design responsive mobile-first websites using compiled CSS and JavaScript plugins.

The accordion utilizes the Collapse JavaScript plugin internally. By setting data-bs-parent to match the parent container ID, opening one accordion pane automatically closes the others.

No, jQuery is no longer a dependency for Bootstrap starting from version 5. All JavaScript plugins are rewritten in vanilla ES6 JavaScript.
