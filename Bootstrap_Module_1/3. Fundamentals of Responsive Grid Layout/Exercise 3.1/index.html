Responsive Three-Column Grid

Stacks vertically on mobile, 2 per row on tablets, and 3 per row on desktops.

Column 1

This column takes up full width (col-12) on mobile screens, half width (col-md-6) on tablet screens, and one-third width (col-lg-4) on desktop screens.

Column 2

Bootstrap handles this wrapping dynamically. On tablet screens, this column sits next to Column 1, while Column 3 wraps underneath them.

Column 3

On desktop screens, all three columns fit in a single row side-by-side because the sum of the columns (4 + 4 + 4) equals 12.
