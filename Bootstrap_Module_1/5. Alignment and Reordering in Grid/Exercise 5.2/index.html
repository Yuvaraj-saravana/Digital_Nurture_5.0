Column Reordering

Using order-md-1 and order-md-2 to reorder elements.

Column A (Text Content)

This is Column A. In the HTML code, it is written first. On mobile screens, it appears on top. On desktop (medium and up) screens, it shifts to the right because of order-md-2.

Column B (Visual Content)

This is Column B. In the HTML code, it is written second. On mobile screens, it appears on the bottom. On desktop screens, it shifts to the left because of order-md-1.
