Responsive Sidebar Demo

A sidebar layout that is completely hidden on mobile viewports but appears on tablets and above.

Sidebar Menu
  • Home Link

  • Help Section

Main Content Block

This main content block is set to col-12 by default (taking full width on mobile) and transitions to col-md-9 on tablets/desktops to leave space for the sidebar.

On mobile screens, you will not see the sidebar column at all because of the d-none helper class. It is only rendered when the viewport expands to 768px (medium screen width) matching the d-md-block class on the sidebar wrapper.
