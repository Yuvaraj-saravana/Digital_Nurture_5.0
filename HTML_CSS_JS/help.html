Help Center & User Guide

Welcome to the user guide for the Local Community Event Portal. Below, we explain the main features of our website and how you can interact with it.

1. Browsing Events

All upcoming community events are displayed dynamically as cards in the **Upcoming Events** section. You can filter these events using the category selection dropdown, or search for events by typing keywords into the search box. The system updates the list dynamically as you type or change filters.

2. Registering for Events

  • From Event Cards: Simply click the "Register" button on any available event card. An alert will confirm your registration, and the remaining seat counter will decrease automatically.
  • From the Registration Form: Scroll down to the Event Registration Form, fill out your full name, email address, select your preferred event from the dropdown options, and click "Submit Registration". Our system validates the inputs, simulates a server check delay, and confirms registration successfully.

3. Key Developer Features

This project demonstrates a variety of core frontend development concepts:

  • Closures: Used to manage session-specific registration data cleanly without polluting the global scope.
  • jQuery Animations: The interactive demo boxes in the Contact section showcase the practical difference between CSS visibility: hidden and display: none in the page layout.
